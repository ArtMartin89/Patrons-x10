# Лендинг "Х10 Патроны"

Это презентационный лендинг для привлечения патронов в сообщество Х10. Лендинг реализован на Next.js с использованием TypeScript, Tailwind CSS и Framer Motion.

## Особенности

- Адаптивный дизайн для всех устройств
- Плавные анимации при скролле
- Загрузка контента из локального JSON-файла
- Презентационный режим (без интеграции с платежными системами)
- Поддержка темной/светлой темы

## Структура проекта

```
landing/
├── src/
│   ├── app/                 # Страницы приложения (App Router)
│   ├── components/          # Компоненты для каждого блока
│   ├── data/               # JSON-файл с контентом
│   └── lib/                # Вспомогательные функции
├── public/                 # Статические файлы
├── package.json            # Зависимости и скрипты
└── README.md               # Документация
```

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите приложение в режиме разработки:
```bash
npm run dev
```

Приложение будет доступно по адресу [http://localhost:3000](http://localhost:3000)

## Сборка для продакшена

```bash
npm run build
```

## Изменение контента

Все текстовые данные находятся в файле `src/data/content.json`. Для изменения контента отредактируйте значения в JSON-файле. Важно не менять ключи, чтобы не сломать отображение.

## Технический стек

- **Next.js 14+** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion** (анимации)
- **React 18**

## Назначение файлов

- `src/app/page.tsx` - Главная страница, координирует отображение всех компонентов
- `src/lib/contentLoader.ts` - Загрузка данных из JSON-файла
- `src/components/` - Компоненты для каждого блока лендинга:
  - `HeroSection.tsx` - Блок приветствия
  - `ProblemSection.tsx` - Блок о проблеме
  - `ConceptSection.tsx` - Блок с концепцией
  - `RecurringLogicSection.tsx` - Блок о логике регулярности
 - `TrustSection.tsx` - Блок о надежности
  - `ValuePropositionSection.tsx` - Блок о ценности
  - `TiersSection.tsx` - Блок с уровнями участия
  - `VisionSection.tsx` - Блок с видением
  - `FooterSection.tsx` - Футер

## Примечание

На данном этапе кнопки "Стать патроном" и выбор тарифов не ведут на платежный шлюз. При нажатии на них появляется модальное окно с текстом "Программа запускается в ближайшее время. Оставьте контакт".
